Exolve Editor

Exolve Editor, as the name suggests, is an editor for files used by the excellent Exolve interactive crossword player. An exolve file consists of two parts:
  • A crossword, defined in a structured text format (see the Exolve README for details).
  • A template, one of two files that ship with exolve, either exolve-m.html or exolve.html.
The crossword is merged into the template to create a standalone HTML file that can be loaded into your browser to solve the crossword interactively, or hosted on your website to present your readers with an interactive crossword.

Crosswords

To load a crossword, click
  • Load Exolve: Extracts the crossword from an existing exolve file. This is useful for, e.g., moving a crossword from an old exolve file and merging it with the latest version of the template.
  • Import QXW: Loads a QXW save file and converts it into exolve format. Note that
    • Only the rectangular format is supported
    • Clues will be generated from the grid, in the form "1. ANSWER", since QXW is purely a grid creation app and does not support clues
    You can add clues directly within this app, but it's probably more convenient to save the converted file and edit it in your favourite text editor.

Templates

Once your crossword is ready, it needs to be merged with an exolve template. Download either exolve-m.html or exolve.html to your computer, then click Load Exolve Template to load it into the Template tab. (If you are not sure what the difference between the two files is, you probably want exolve.html.)

Saving the completed file

When you are read to save, click on Save Exolve. This will merge the contents of the Crossword and Template tabs and create an HTML file. To load a crossword, click

Click 'Load QXW' to load a qxw file. The file will be converted to exolve. After that, you can 'Copy to clipboard' to copy the generated exolve text, and paste it into your html file, or 'Load exolve template' followed by 'Save exolve' to merge your crossword into an exolve html file.

Exporting to other formats

While this app is primarily focused on exolve files, the contents of the Crossword tab can be converted to other formats. Currently the only other supported format is the crossword markup used by the /r/crosswords subreddit; open an issue if you would like some other export format, or see my pangrid command line tool for a more general conversion utility. Clicking on Output Reddit will convert the current crossword to reddit markup format, and write it to the Output tab, where it can be copied or saved.
